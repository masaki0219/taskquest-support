# TaskQuest サポートページ

TaskQuest は、毎日のタスクをクエスト形式で進めながら、ゲーム感覚で習慣化をサポートするアプリです。  
AI によるタスク生成、ペット育成、クエストステージなど、続けやすい仕組みを組み合わせています。

## 📘 アプリの主な機能
- AI によるデイリー / ウィークリータスク自動生成  
- コインや経験値が獲得できるゲーム型システム  
- ペット育成とクエストステージ進行  
- 過去のタスク履歴の自動記録  
- シンプルで直感的な UI
